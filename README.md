# NetPolyFills
ES6 support for Netscape Navigator 9.0.0.6 starting off from Windows 7 and later!
dependencies:
  Microsoft Edge (with the extension, Netscape 9.0.0.6 simply offloads JS to Edge, which is why NetPolyFills can run only on Windows 7 and later.)
  PowerShell ISE (reccomended)
method:
  1)download and extract the zip.
  2)unless you already have it, use the installer to get Netscape 9.0.0.6
  3)run the PowerShell script
  4)install the .xpi
  5)enjoy!
note: this is not an open-source project.
